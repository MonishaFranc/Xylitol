Junior Software Developer

Currently we are looking for passionate and talented SOFTWARE DEVELOPERS to join our team as Junior Software Developer. As a Junior Software Developer, you will work on real-world projects and gain hands-on experience working with cutting-edge technologies.

Responsibilities

  • Collaborate with the development team to build software applications
  • Write clean, efficient, and maintainable code
  • Participate in code reviews and contribute to team knowledge sharing
  • Test and debug software applications

Requirements

  • Currently pursuing a degree in Computer Science or a related field
  • You are passionate about web development and have gained initial experience with web technologies
  • Strong problem-solving skills and attention to detail
  • Experience with one or more programming languages (JavaScript, Angular, React, etc.)
  • Experience with Git and version control
  • You are flexible and feel comfortable in an agile environment
  • You prefer working independently on a project to detailed instructions

Benefits

  • Hands-on experience working on real-world projects
  • Mentorship from experienced software developers
  • Opportunity to work with cutting-edge technologies
  • Flexible work hours and remote work options
